# C# 계산기 프로젝트

이 프로젝트는 NetBeans 환경에서 C#으로 개발한 기본적인 계산기입니다.

## 기능

- 사칙연산 (덧셈, 뺄셈, 곱셈, 나눗셈)
- 연산자 우선순위 처리
- 실시간 결과 표시

## 아키텍처

프로젝트는 간단한 객체 지향 프로그래밍(OOP) 원칙을 따릅니다:

- **모델 (Model)**: 계산의 로직을 처리합니다.
- **뷰 (View)**: 사용자 인터페이스(UI)를 담당합니다.
- **컨트롤러 (Controller)**: 모델과 뷰 사이의 상호작용을 관리합니다.

## 사용된 기술

- **C#**: 주 프로그래밍 언어
- **NetBeans**: 개발 환경(IDE)
